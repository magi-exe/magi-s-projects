1. Use Command Prompt (Secure Hide) 
This method marks a file or folder as a System File, which adds a second layer of concealment. 
Super User
Super User
 +1
Open the Start menu, type cmd, and press Enter.
Navigate to your file's location using the cd command (e.g., cd Desktop\MyFolder).
Run the following command to hide the file:
attrib +h +s +r "FileName.ext"
+h: Sets the Hidden attribute.
+s: Sets the System attribute.
+r: Sets the Read-only attribute (prevents accidental deletion).
To unhide: Run attrib -h -s -r "FileName.ext".

2. Use PowerShell 
PowerShell offers a similar programmatic way to toggle attributes. 
Right-click the Start button and select Windows PowerShell (Admin).
To hide a folder or file, run:
Set-ItemProperty -Path "C:\Path\To\File" -Name Attributes -Value Hidden
To unhide: Run the same command but change the value to Normal:
Set-ItemProperty -Path "C:\Path\To\File" -Name Attributes -Value Normal
